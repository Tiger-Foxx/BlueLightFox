🚀 PREMIERS PAS

• L'application démarre automatiquement après l'installation

• Cliquez sur l'icône pour ouvrir le panneau de configuration
• Essayez différents modes pour trouver celui qui convient le mieux à vos yeux

💡 CONSEILS D'UTILISATION

• Mode "Nuit" : Idéal pour une utilisation nocturne, réduit considérablement la lumière bleue
• Mode "Lecture" : Optimisé pour la lecture prolongée de textes
• Mode "Personnalisé" : Ajustez manuellement les niveaux RGB selon vos préférences
• Activation programmée : Configurez l'application pour s'activer automatiquement le soir
